# ADIFAL — Website

Site oficial da **ADIFAL — Inteligência Agrícola**.

Landing page para apresentação do software de gestão de gado e download da versão beta.

## Estrutura

```
GadoWeb/
├── index.html              ← Página principal
├── css/
│   └── style.css           ← Estilos
├── js/
│   └── main.js             ← Interações e animações
├── assets/
│   ├── images/             ← Logótipo, screenshots, etc.
│   └── downloads/          ← ZIP da versão beta
├── pages/                  ← Páginas adicionais (futuro)
└── README.md
```

## Como Usar

1. Colocar o logótipo em `assets/images/logo.png`
2. Colocar o ZIP da app em `assets/downloads/GadoApp-Beta.zip`
3. Abrir `index.html` no browser

## Hosting

Site estático — pode ser hospedado em:
- **GitHub Pages** (grátis)
- **Netlify** (grátis)
- **Vercel** (grátis)
- Qualquer servidor web (Apache, Nginx, etc.)
